**The Maze** is an exciting **2D pixel art game** built entirely in **Java** using **Swing** and **AWT** for the graphical user interface. In this game, players navigate through a labyrinth filled with enemies, traps, and treasures, while trying to find their way to the exit. Along the way, they can visit a **shop** to purchase upgrades like health potions, better weapons, and enhancements. ## üîë Features

**The Maze** offers a variety of gameplay mechanics that keep players engaged throughout their journey. Below are the key features:

- **Labyrinth Generation**: Each time the game is played, the maze is procedurally generated, ensuring that players experience a new challenge every time they play. The labyrinth is filled with various obstacles, rooms, and hallways that need to be explored and conquered.
- **Combat**: The game features a variety of enemies, each with different behaviors and attack patterns. Players must fight enemies using simple attack controls while avoiding damage. As players progress, the enemies become tougher and more numerous.
- **Shop Mechanics**: A shop system allows players to purchase upgrades such as health potions, better weapons, and other consumables. Currency is earned by defeating enemies and completing objectives within the maze.
- **Pixel Art Style**: The game uses a pixel art style, which not only gives it a retro feel but also contributes to its charm. Each character, enemy, and environmental element has been designed with care to create an authentic 2D game aesthetic.
- **Simple Yet Engaging Gameplay**: The game is designed to be easy to pick up but hard to master. Players need to make strategic decisions as they navigate the maze, decide when to fight, and when to run or heal.
- **GUI with Java Swing and AWT**: The entire user interface is designed using Java Swing and AWT, making it highly responsive and customizable. This allows for a smooth, consistent user experience.
